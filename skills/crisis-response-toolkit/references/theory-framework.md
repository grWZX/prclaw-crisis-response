# 理论框架 —— crisis-response-toolkit

本 Skill 全部理论来源于课程课件《第十讲：危机传播管理》（北京交通大学传播系）。


## 一、危机本质论（§1，第4-10页）

| 核心命题 | 课件出处 | 对本Skill的指导 |
|---|---|---|
| 危机不是非常态，而是现代性的常态后果 | 第5页 | 建立常态化响应机制 |
| 危机本质上是一种威胁性的形势、情境和状态 | 第6页 | 关注"情境"而非仅"事件" |
| "化危为机"是一种错误想象 | 第9页 | 不以"转危为机"为评价标准 |
| 危机提供最强信号，促使组织再造常态秩序 | 第9页 | 回应应包含"系统整改"承诺 |


## 二、"事实-价值"模型（§2，第11-15页）

**核心命题**：危机传播管理 = 事实层面（真相查证+利益互惠）+ 价值层面（信任重建+意义分享）

### 2.1 事实层

| 任务 | 关键动作 | 输出体现 |
|---|---|---|
| 真相查证 | 告知已知事实、说明调查进展 | 声明中必须有"已知事实"段落 |
| 利益互惠 | 提出补偿方案、明确责任追究 | 必须有"整改措施"或"补偿方案" |

### 2.2 价值层

| 任务 | 关键动作 | 输出体现 |
|---|---|---|
| 态度表态 | 道歉、致歉 | 必须有"道歉"或"致歉" |
| 情感共情 | 承认公众情绪 | 必须有"理解""担忧"等词汇 |
| 信任重建 | 具体承诺+时间节点 | 必须有"将在X月X日前公布" |

### 2.3 三阶段管理

| 阶段 | 核心任务 | 在本Skill中的体现 |
|---|---|---|
| 预警 | 事实监测+价值评判 | 输入阶段的情感指数判断 |
| 处置 | 告知真相+重塑信任 | 声明/Q&A生成 |
| 恢复 | 重建事实契约+再造价值体系 | 长期承诺与整改时间表 |


## 三、危机管理五路径（§3，第17-32页）

| 路径 | 核心策略 | 适用场景 | 关键动作 |
|---|---|---|---|
| **告知** | 讲真话、第一时间反应 | 事实逐步明朗、公众信息饥渴 | 不说谎、不躲避、不出丑 |
| **疏导** | 适度承诺、引领议题 | 舆论混乱、谣言四起 | 成为第一信源、引入意见领袖 |
| **转换** | 补偿利益相关者、共担风险 | 对抗性强、责任明确 | 旁观者→共担风险的主人 |
| **顺应** | 倾听、道德加法、关爱弱者 | 情绪高涨、道德谴责 | "我已经听到" |
| **引领** | 统一口径、休戚与共 | 内部混乱、长期信任重建 | 许诺共处共持的价值共同体 |

### 路径组合原则

| 危机类型 | 推荐路径组合 | 理由 |
|---|---|---|
| 事实未明型 | 告知 + 疏导 | 先沟通不确定性，再引领议题 |
| 责任明确型 | 转换 + 顺应 | 承认责任、补偿受损方、表达歉意 |
| 情绪高涨型 | 顺应 + 疏导 | 先共情倾听，再引领理性讨论 |
| 内部混乱型 | 引领 + 告知 | 先统一内部口径，再对外告知 |
| 长期信任重建 | 引领 + 转换 | 价值共同体许诺 + 利益相关者补偿 |


## 四、新闻发布三原则（§4，第37-41页）

| 原则 | 核心要求 | 检查标准 |
|---|---|---|
| **时间性** | 第一时间反应；态度话语优先 | 30分钟内表态；先道歉再说理 |
| **主体性** | 掌握话语权；多元话语平衡 | 有标准故事；引入意见领袖 |
| **解释力** | 持续提供有生命力的信息；体验式透明 | 使用"是/故事/比喻/视觉"修辞 |


## 五、媒体应对技巧（§4，第42页）

> "坦诚，说真话；对人，说人话；有效，说对话；修辞，说巧话；选择，说'糊'话。"

### 逐条解读

| 原则 | 含义 | 在声明中的体现 |
|---|---|---|
| 坦诚，说真话 | 不撒谎，不隐瞒 | 所有事实陈述必须可验证 |
| 对人，说人话 | 避免官话套话 | 用"对不起"替代"深表遗憾" |
| 有效，说对话 | 针对性回应 | 回应公众最关心的议题 |
| 修辞，说巧话 | 适当的修辞技巧 | 使用"认同"策略 |
| 选择，说"糊"话 | 不确定时沟通不确定性 | "正在调查中" |


## 六、亚里士多德修辞三要素（补充，第四讲）

本 Skill 中 Ethos/Pathos/Logos 的评分标准源于第四讲《公关的修辞与说服》：

| 要素 | 定义 | 在危机声明中的体现 |
|---|---|---|
| **Ethos（可信度）** | 说话者的身份、资质、责任、善意 | 明确责任主体、体现善意、给出具体动作 |
| **Pathos（情感共鸣）** | 阅听人的情感处境 | 承认公众情绪（愤怒/担忧/悲伤） |
| **Logos（逻辑事实）** | 内容本身的事实、证据、逻辑 | 具体时间/地点/数据、区分已知与调查中 |
