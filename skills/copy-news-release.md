# 公关文案与新闻发布 Skill

目标：当用户需要撰写企业声明、道歉信、新闻通稿、新闻发布会口径、明星/公众人物回应时，用“修辞-说服-认同”框架生成可审校文本。

理论来源：
- 第四讲“公关的修辞与说服”：亚里士多德修辞三要素，ethos 指向说话者可信度，pathos 指向阅听人的情感与处境，logos 指向内容、证据与推理。
- 肯尼斯·伯克新修辞：修辞不是只追求压服，而是通过共同情感、共同处境、共同符号建立认同，消解组织与公众之间的隔阂。

工作流程：
1. 先识别场景：企业声明、危机道歉、新闻通稿、新闻发布会 Q&A、明星回应、政府发布。
2. 明确利益相关者：受害者/消费者/员工/媒体/监管者/粉丝/公众。分别写出他们最关心的事实、情绪和价值。
3. 建立三要素：
   - Ethos：说话者身份、责任边界、可信动作、第三方背书。
   - Pathos：承认对方感受，避免傲慢、反问、说教、甩锅。
   - Logos：事实时间线、证据、处置进展、可验证承诺。
4. 用伯克“认同”补强：找到组织与公众共享的底线，如安全、尊重、公平、透明、专业、公共利益。
5. 输出文本时必须包含：开场态度、事实说明、责任表达、处置动作、后续承诺、联系方式或更新机制。

声明写作禁忌：
- 不用“个别员工”“临时工”“网友误解”等先行甩锅表达。
- 不在事实未明时过度承诺；不在情绪高点使用威胁式律师函。
- 不把“抱歉造成困扰”当作唯一歉意；要说明为什么抱歉、对谁抱歉、如何改。
- 新闻发言人口径中，未知信息要说“仍在核实”，不要编造确定性。

输出格式：
1. 修辞诊断：Ethos / Pathos / Logos / 认同点
2. 文案正文：按用户指定体裁生成
3. 媒体追问 Q&A：至少 5 个尖锐问题及回应
4. 风险复核：列出可能引发二次舆情的句子并改写
